# VWAP Bounce / Trend Following Strategy
## Used by: Brian Shannon (AlphaTrends), Larry Jones (Stock Up)

### Overview
VWAP = fair value. Price above VWAP = bullish. Pullback to VWAP that bounces and reclaims it = high-probability long entry. Brian Shannon pioneered anchored VWAPs.

### Core Rule
**Price > VWAP = only take longs. Price < VWAP = avoid longs.** This one filter eliminates many bad trades.

### Anchored VWAP Placement
Anchor VWAP from:
- **Swing highs:** Price pulls back to AVWAP = support
- **Swing lows:** Price pulls back to AVWAP = support
- **Gap days:** Major gaps create AVWAP support
- **Earnings days:** Strong earnings gap days = AVWAP support

### Entry: VWAP Bounce
1. Stock in uptrend, above VWAP
2. Pulls to touch VWAP/AVWAP
3. Bounces with volume
4. Closes above VWAP
5. Enter close or next open

### Entry: VWAP Breakout
1. Stock was below VWAP (weak)
2. Breaks above VWAP on high volume
3. Enter on close above or pullback to VWAP as support

### Stop Loss
- Below VWAP (for bounce entry)
- Below the swing low (safer)
- Tight: below entry candle low

### Targets
- Next resistance level
- Prior high
- Measured move = size of last VWAP swing

### Confluence
Multiple AVWAPs at same level = strongest support (Martin Luk's proven method)

### Adapting for $25-100k
- SPY/QQQ hourly VWAP bounces
- Individual stocks with good volume
- Set AVWAPs on Sunday for the week
