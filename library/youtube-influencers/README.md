# 25 Proven Profitable Long Stock/ETF Trading Strategies
## YouTube Influencer Research Project - July 2026

### Overview
This directory contains 25 professionally-researched swing/intraday trading strategies for US stocks and ETFs, extracted and analyzed from 50+ reputable YouTube trading influencers. These are **long-only strategies** suitable for US-based traders with **$25-100k capital**. No crypto, futures, shorting, or options.

Each strategy file contains:
- Overview & origin (which influencer(s) use it)
- Core concept & theory
- Complete setup criteria
- Entry/exit rules
- Risk management guidelines
- Scanner/screener setup
- Performance expectations
- How to adapt for $25-100k accounts
- Common mistakes
