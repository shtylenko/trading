# TREND_PULLBACK_DETERMINISTIC — 0.4.0 (SMA50 pullback)

No agent loop. Auto-arm scanner plan on the reclaim bar; engine owns fills/exits.

## Why this version (last iteration)

| Version | Idea | n30 effR |
|---|---|---|
| 0.2.0 | EMA20 + tight admission + structure targets | **−0.19** (best so far) |
| 0.3.0 | Same + reclaim_close + 1R/2R | −0.29 (reject) |
| **0.4.0** | **SMA50 pullback** (different level), keep 0.2 construction | TBD |

Hypothesis: EMA20 pullbacks on liquid names are overcrowded / low edge; **deeper
SMA50** reclaims in an uptrend are fewer and more Lance/swing-authentic.

## Rules

**Admission (quality from 0.2.0, applied to SMA50):**
- Uptrend: close > SMA50 & SMA200; SMA50 rising; SPY > SMA50
- Pullback: low ≤ SMA50 (true tag), ≥1 close ≤ SMA50, depth 4–18%, 3–15 bars
- Reclaim: close back above SMA50, extension ≤4%
- Cooldown 12 bars

**Construction (0.2.0 winner):**
- Trigger = reclaim-day **high**
- Stop = pullback low − 0.15×ATR
- T1 = prior swing high; T2 = T1 + measured move
