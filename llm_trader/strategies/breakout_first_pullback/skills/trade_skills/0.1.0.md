# BREAKOUT_FIRST_PULLBACK_DETERMINISTIC — 0.1.0

No agent loop. On the setup bar with `scanner_plan`, emit `ARM_BUY_STOP` with
scanner trigger/stop/ATR/gap/expiry. All other bars: `OBSERVE`. Engine owns
fills, targets, gap cancel, expiry, and flatten.

## Setup (scanner)

1. Multi-week base (15–60 bars, range 4–18%)
2. Breakout: green close clears base high with volume ≥1.3× 20d avg
3. First pullback tags base high / breakout level; close holds above it
4. Trigger = setup high; stop under pullback low; T1 post-break swing high;
   T2 = T1 + base height
5. Optional: SPY > SMA50; name above SMA50 with SMA50 rising

## Validity

v0.1.0 baseline only — no profitability claim. Smoke → n30 → multi-year before
promotion.
